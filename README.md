Landing Page Project

Create landing page which is interactive .Adding sections will add it in the navigation bar and highlight the active  section which appear in window.

Getting Started
Starter code taken from udacity repository. 
The starter project has some HTML and CSS styling to display a static version of the Landing Page project. Project is to convert this starter project from a static project to an interactive one. This will require modifying the HTML and CSS files, but primarily the JavaScript file.

Prerequisites

You need to install a web browser such as google chrome
and a text editor such as VS code
